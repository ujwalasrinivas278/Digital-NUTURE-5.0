# SQL Exercise - Functions

## Exercise 7 - Return Data From Scalar Function

### Objective

To understand and implement Scalar Functions in SQL for performing reusable calculations.

### Problem Statement

Create a scalar function named `GetAnnualSalary()` that accepts a monthly salary and returns the annual salary. Use the function within a SELECT statement to display employee salary details.

### SQL Concepts Used

- CREATE FUNCTION
- RETURN Statement
- Scalar Functions
- Parameters
- SELECT Statement

### Conclusion

Scalar functions provide reusable business logic and calculations within SQL queries. They improve code modularity, readability, and maintainability while reducing repetitive calculations.
