# SQL Exercise - Advanced Concepts

## Exercise 1 - Ranking and Window Functions

### Objective

To understand and implement SQL Window Functions for ranking and numbering records based on salary.

### Problem Statement

Create an Employee table and use the following window functions:

- RANK()
- DENSE_RANK()
- ROW_NUMBER()

to rank employees according to their salary in descending order.

### SQL Concepts Used

- Window Functions
- RANK()
- DENSE_RANK()
- ROW_NUMBER()
- ORDER BY
- SELECT Statement

### Conclusion

Window functions provide powerful analytical capabilities in SQL. They help rank, number, and analyze rows without grouping data, making them essential for reporting and business analytics.
