# SQL Exercise - Index

## Exercise 1 - Create Index

### Objective

To understand the concept of indexing in SQL and improve query performance on frequently searched columns.

### Problem Statement

Create a Customers table and add an index on the CustomerName column. Execute a query using the indexed column to retrieve customer details.

### SQL Concepts Used

- CREATE INDEX
- SELECT Statement
- WHERE Clause
- Query Optimization
- Database Indexing

- Exercise1_CreateIndex.sql

### Conclusion

Indexes improve the speed of data retrieval operations by reducing the amount of data scanned during query execution. They are essential for optimizing database performance, especially for large datasets.
