## Program Overview

**Digital Nurture 5.0** is Cognizant's internal upskilling initiative designed to elevate DotNet Full Stack Engineers (FSE) with a comprehensive, hands-on curriculum spanning frontend, database, and backend technologies. The program is self-paced, driven by open-source learning resources, and is expected to be completed in **3 weeks**.

\---

## Skills Covered

|Domain|Technologies|
|-|-|
|Frontend|HTML5, CSS3, JavaScript, Bootstrap 5, jQuery|
|Database|ANSI SQL using MySQL|
|Backend|C# (.NET 8 / C# 12), ADO.NET|

## Module 1 — Frontend Fundamentals

### HTML5

**Overview:** Covers the essential concepts, elements, and best practices of HTML5, including modern APIs such as Web Storage, Geolocation, and client-side database handling.

**Learning Objectives:**

* Set up the HTML5 development environment
* Apply semantic HTML5 tags for structured content
* Build navigation and page references
* Handle DOM events and user interactions
* Implement advanced HTML5 form features
* Use Web Storage (session \& local) and Web SQL Database

## Module 3 — C# and ADO.NET

**Overview:** In-depth exploration of modern C# (up to version 12) and ADO.NET within the .NET 8 ecosystem — covering OOP, async programming, multi-threading, secure coding, and database access patterns.

**Learning Objectives:**

* Install and configure .NET SDK and Visual Studio / VS Code
* Write C# programs using correct syntax, data types, and control flow
* Apply OOP principles: classes, objects, inheritance, polymorphism, encapsulation
* Use modern C# 12 features: primary constructors, records, pattern matching, list patterns
* Handle exceptions robustly with try-catch-finally and null safety operators
* Use collections (List, Dictionary, Queue, Stack) and write LINQ queries
* Implement async/await patterns and understand multi-threading with synchronization
* Read/write files and serialize JSON using System.Text.Json
* Apply secure coding practices against XSS, SQL injection, CSRF, and broken authentication
* Connect to databases using ADO.NET's Connection, Command, DataReader, DataAdapter, DataSet, and DataTable classes

## Exercise Workflow

All hands-on exercises are hosted in a public GitHub repository. Follow these steps:

### 1\. Access the Repository

```
https://github.com/seshadrimr/Digital-Nurture-DotNetFSE
```

Navigate to: `DotNet FSE` → `Upskilling`

### 2\. Download Exercises

Click the **Download** button on the repository page to get the exercise files.

### 3\. Solve the Problems

* Read each problem statement carefully
* Apply the concepts covered in the corresponding module
* Think through solutions before coding them

### 4\. Self-Evaluate

* Review your solutions against the problem criteria
* Compare with any provided hints or sample solutions
* Reflect on areas of improvement

### 5\. Submit Solutions

1. Organize solutions in a dedicated folder
2. Create a **public repository** on your personal GitHub account
3. Upload your solution folder
4. Share the repository URL with your Point of Contact (POC) on demand

### 6\. Getting Help

* Collaborate with peers for problem-solving support
* Refer back to the learning reference links in this handbook
